1. Initial Setup
Download my code (ZIP file) and extract it to a folder.

Download and install Node.js version v14.17. https://nodejs.org/dist/v14.17.0/node-v14.17.0-x64.msi

Set node.exe as the default program for opening .js files.

2. LoopMIDI Configuration
Download and install the LoopMIDI software. https://www.tobias-erichsen.de/software/loopmidi.html

Add a virtual device named resolume2fire.

3. Resolume Preparation
Launch Resolume.

In the MIDI settings:

Set AKAI Fire input as the input device.

Set resolume2fire as the output device.

Optionally, copy and apply my default mapping from the provided .xml file.

Save the settings and close Resolume.

4. Launching the Setup
Connect the AKAI Fire and wait for it to boot.

Run my code by clicking the green fire2resolume icon.

If a window appears with the word “Resolume,” launch Resolume Arena.

Everything should now be working.

Inside the folder, you’ll find:

The XML mapping file.

A PNG image showing the configuration settings.
